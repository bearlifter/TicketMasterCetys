# TicketMasterCetys
Repositorio para proyecto Desarrollo Ágil


Para correr solo es necesario entrar a las carpetas 

client y server y correr

`npm install` y `npm run dev`

en el .env del backend es necesario tener esto:

`PORT=5000
MONGODB_URI=
JWT_SECRET=
NODE_ENV=
`
